This is the prototype for the Report funding progress (Report) service. The service gives grant recipients a way to report on monitoring and evaluation data.

Report aims to replace Submit monitoring and evaluation data (Submit) by iteratively moving away from users inputting information into a spreadsheet to form-based question flows and a task list.
